# sepet-uygulaması-
🛒 Sepet Uygulaması

Python ile geliştirilmiş basit bir konsol sepet uygulamasıdır.

🚀 Özellikler

- Ürün ekleme
- Sepetteki ürünleri listeleme
- Toplam tutarı hesaplama
- Menü tabanlı kullanım

🛠️ Kullanılan Teknoloji

- Python

📚 Bu projede öğrendiklerim

- Listeler ("list")
- Döngüler ("while", "for")
- Koşullar ("if", "elif", "else")
- Kullanıcıdan veri alma ("input")
- Toplam hesaplama
